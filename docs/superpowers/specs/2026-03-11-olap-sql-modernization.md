# olap-sql 现代化迭代设计文档

**日期：** 2026-03-11  
**目标：** 分阶段对 olap-sql 进行现代化维护和文档建设，保持向后兼容

---

## 背景

olap-sql 是一个 Go 的 OLAP 查询 SQL 生成库，支持 metrics、dimension、filter 的组合查询，自动生成适配多种数据库的 SQL。当前 Go 1.18，依赖较旧，文档不够完善。

## 目标

1. **现代化维护** — Go 升到 1.24，使用新语法，依赖分阶段升级
2. **文档建设** — 使用者文档 + 贡献者文档

## 原则

- 分阶段渐进，**每个 Phase 切换前等用户确认**
- 每次改动单独 PR，保持可回滚
- Breaking change 单独 PR + CHANGELOG
- 语法现代化和依赖升级分开 PR
